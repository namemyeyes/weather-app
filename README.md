# 🌤️ Weather App — Прогноз погоды в реальном времени

Современное веб-приложение для отслеживания погоды, использующее **Open-Meteo API** и **Geocoding API**. Проект выполнен с акцентом на чистый код, производительность и адаптивность.

**🔗 [Живая ссылка на проект](https://namemyeyes.github.io/weather-app/)**

---

## 🚀 Основные возможности

* **Умный поиск**: Автодополнение названий городов при вводе (Geocoding API).
* **Гибкие настройки**: Переключение систем измерения (Celsius/Fahrenheit, km/h / mph, mm/inch).
* **Детальный прогноз**: 
    * Текущая погода с динамическими фонами.
    * Почасовой прогноз.
    * Прогноз на 7 дней вперед.
* **Full Responsive**: Адаптивный дизайн, оптимизированный под мобильные устройства.

---

## 🛠️ Технологический стек

* **HTML5**: Семантическая верстка.
* **CSS3**: Использование переменных (CSS Variables), Flexbox, Grid Layout и Media Queries (в единицах `rem`).
* **JavaScript (ES6+)**: 
    * Асинхронные запросы (`async/await`, `fetch`).
    * Динамический рендеринг DOM.
    * Обработка ошибок (Error Handling) с визуальным фидбеком для пользователя.
* **API**: [Open-Meteo](https://open-meteo.com/).
